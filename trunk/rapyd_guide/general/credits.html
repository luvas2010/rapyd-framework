Credits

Code Igniter

Code Igniter was developed by Rick Ellis, who in his other life is CEO of Ellislab, Inc. The core framework was written specifically for this application, while many of the class libraries, helpers, and sub-systems borrow from the code-base of ExpressionEngine, a Content Management System written by Rick Ellis and Paul Burdick.

A hat tip goes to Ruby on Rails for inspiring us to create a PHP framework, and for bringing frameworks into the general consciousness of the web community.

The Code Igniter logo and icons were created by Rick Ellis.

The pull-down table of contents was created with the use of the moo.fx library.

Rapyd Library for Code Igniter

Important:  Rapyd is not an "official" or "approved" addon/library for CI. It's just a project coming from CI fans/community guys.

Rapyd was developed by Felice Ostuni freelance at skinsoftware.com, from italy.
Since version 0.8 Thierry Rey joins development with significant enhancements to many components.

The layout, navigation and content organization was entirely taken/insipred from code igniter user guide. The highlighter used for code is a javascript replacement for textareas SyntaxHighlighter by dreamprojections.
