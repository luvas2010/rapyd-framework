Server Requirements

  • PHP version 4.3.2 or newer
  • A Database. Rapyd is tested on MySQL (4/5) and Postgre, however supported databases should bee the sames of CI (MySQL, MySQLi, MS SQL, Postgre, Oracle, SQLite, and ODBC).
  • Rapyd 0.9.5 Require Code Igniter 1.5.1
