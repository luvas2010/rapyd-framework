Rapyd Library at a Glance

Cruds in few minuts

If you like Code Igniter, but you need to boost your development (at least for backends) Rapyd Library is a good choise. Its goal is to enable you to develop projects much faster than you could if you were writing code from scratch, and to leave you decide when is time to use classic mvc approach, or to build a complete and complex crud in 10 line of code in few minutes directly in a controller without models or views. His syntax is concise, easy to be mantained.

Rapyd is Open Source

Rapyd Library is licensed under LGPL open source license so you can use it however you please.

Rapyd Runs on PHP 4

Rapyd is written to be compatible with PHP 4.

Note: Rapyd Library will run on PHP 5. It simply does not take advantage of any native features that are only available in that version.

Rapyd is Light Weight

The core system requires only a few very small libraries. It has the same philosophy of CI, components are loaded dynamically upon request (as CI libraries). Based on your needs for a given process, so the base system is very lean and quite fast.

It include TinyMCE that has a lot of files, but is really the better and smaller wysiwyg replacement for textareas used by most of cms (wordpress, drupal, joobla, etc.)

Rapyd Generates Clean URLs

components generated urls are semantic, clean and search-engine friendly:

www.your-site.com/controller/function/orderby/userid/desc
www.your-site.com/controller/function/search_uri/name/jhonny

Component Based approach

With rapyd you can think/develop by "components". So, if you need to build a "filtered grid"

instead of:
- " I will develop my model.. create my custom sql, iterate my custom view, sniff Post, intercept actions.."
you can think:
- " I need a DataFilter joined with a DataGrid."

Rapyd is Extensible

Rapyd can be easily extended by creating new components/classes, using the loader class: //in controllers/models

$rapyd->load->("mywidget");


//you can access ci & rapyd from your classes

class myWidget {
..
function myWidget()
{
 ..
 $this->ci =& get_instance();
 $this->rapyd =& $this->ci->rapyd;
}
..
}

Themes

Rapyd has a theme system which is thought to standardize components layout.
It comes with a "default" folder theme, but you can build your custom theme in few minutes.
