Rapyd URI Class

Important:  It is not a extention of Code Igniter uri class.

Rapyd URI class is an helper used by rapyd components to work with uri.
You can use this reference: $this->rapyd->uri.

URI Keywords

Most of rapyd components are "small applications", their "status" and their "actions" are driven by uri-segments.
So words like "show","modify","search", etc.. are reserved by rapyd.

You can configure (change) these words to prevent conflict with your "segments" (controller's names for example).

Rapyd URI Reference

$this->rapyd->uri (by default is instanced with rapyd, on library load).

Important:  These properties are stored in /system/application/config/rapyd.php

Methods

$this->rapyd->uri->keep_persistence();

Used before rapyd components:

It enable page persistence.
Components "links","actions urls" are rebuilt adding/keeping a pair of segments: gfid/xxxxx where "xxxxx" is a pointer to the session persistence item (see rapyd session class).
See rapyd samples (DF+DG), it is used to keep orderby,offset,search persistence of listing page when you "move to"/"return from" the detail page.
